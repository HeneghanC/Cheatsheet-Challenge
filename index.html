How to use Layout Properties in CSS

Property Use Description
position position: static; The position property allows you to specify the position of elements.
relative position: relative; This sets an element's position as relative to it's default position on the page.
absolute position: absolute; This sets an element's position as relative to its closest positioned parent element.
fixed position: fixed; This pins an element's position to any part of the page. It will remain in view no matter what.
sticky position: sticky; This sticks an element to a defined offset position when the user scrolls its parent container.
z-index z-index: 1; This specifies how far back or forward an element appears on the page when it overlaps other elements.
display display: block; This allows you to control how an element flows vertically and horizontally in a document.
inline display: inline; This causes elements to take up as little space as possible and cannot have manually adjusted height or width.
block display: block; This causes elements to take up the width of their container and can have manually adjusted height and width.
inline-block display: inline-block; This causes elements to have a set width and height. They can appear next to each other and do not take up their entire container width.
float float: left; This property can move elements as far left or right as possible on a web page.
clear clear: right; This property can clear an element's left or right side (or both).
